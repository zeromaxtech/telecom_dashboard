# 📱 India Telecom Subscriber Dashboard

A data analytics dashboard visualizing India's telecom subscriber data from TRAI (Telecom Regulatory Authority of India) for March 2026.

## 🎯 Project Overview

This dashboard provides insights into:
- **Total telecom subscribers** across 22 Indian states
- **Operator market share** (Jio, Airtel, Vodafone, BSNL, MTNL)
- **Urban vs Rural** subscriber distribution
- **State-wise breakdown** of subscriber data

## 📊 Data Source

- **Source**: TRAI Press Release No. 53/2026 (March 2026)
- **Coverage**: 22 Licensed Service Areas in India
- **Metrics**: Wireless subscribers, broadband, wireline data

## 🛠️ Technologies Used

- **Python 3.x**
- **Streamlit** - Interactive web dashboard
- **Pandas** - Data manipulation
- **Matplotlib** - Data visualization
